**# Simple Python Roulette Simulator**

A lightweight, terminal-based Roulette game built with Python. This script simulates a classic European Roulette experience where players can place bets on specific numbers and test their luck against the house.
--------------------------------------------------------------------------------------------------------------------------------------------
**Features**
European Layout: Includes numbers 0-36.
Colorized Interface: Uses stylized terminal output for a better user experience.
Real-time Win/Loss Calculation: Instantly updates your balance based on the spin result.
--------------------------------------------------------------------------------------------------------------------------------------------
**Prerequisites**
To run this program, you need to install the Colorama library to ensure the terminal colors display correctly across different operating systems.
Install it via pip using the following command:
pip install colorama
